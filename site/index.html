🎙️

Push-to-talk + wake-word

Atalho global (⌘⇧H por defeito) ou palavra-chave personalizável («hey hermes», «alexa», …). Captação visual com VU meter e cronómetro de gravação.

🧠

STT local ou cloud

whisper.cpp local (offline, privado) ou OpenAI Whisper API. Mudas a qualquer momento sem reiniciar — instalação automática via Homebrew.

🔊

Voz natural sem internet

Piper (local, ~30 MB por voz) ou ElevenLabs cloud. Catálogo de vozes em PT, EN, ES e mais. Barge-in: interrompe a meio para fazer outra pergunta.

🔌

Multi-agente

Bridge desenhado para qualquer agente OpenAI-compatível. Hoje: Hermes. Brevemente: Claude Code, ChatGPT, Cursor, OpenClaw — todos pela mesma ponte.

⚡

Atalhos para tudo

⌘L limpa, ⌘R reverte erros, ⌘S exporta a conversa, ⌘, abre Definições. Tudo fica num só ecrã pequeno que cabe no canto do monitor.

🔐

Local-first & privado

O bridge corre no teu servidor (não nosso). Token de pareamento via QR ou copy-paste. Sem telemetria, sem analytics, sem contas.

Perguntas frequentes

Funciona sem internet?

Sim, se escolheres Whisper local (STT) + Piper (TTS) + um agente que corra na tua rede. A app, o bridge e o agente podem todos estar offline.

Que agentes suporta hoje?

Hermes (NousResearch) é o único oficialmente suportado neste momento. O bridge é desenhado para qualquer endpoint /v1/chat/completions compatível com OpenAI, então tecnicamente ChatGPT e a maioria dos servidores de LLM open-source também funcionam — falta só o flag --agent= que está documentado em docs/Agents.md.

Que sistemas operativos?

A app desktop é macOS only por enquanto (Apple Silicon + Intel). O bridge corre em Linux (Debian, Ubuntu, Fedora, Arch, openSUSE — qualquer distro com systemd). Windows + Linux desktop ficam no roadmap.

Privacidade?

Zero telemetria. Sem contas. O áudio vai do teu Mac directamente para o teu bridge, que está no teu servidor. Se escolheres Whisper local + Piper local, nada chega à internet pública. Se escolheres OpenAI ou ElevenLabs, vai só esse pedaço — a app não acrescenta nada por cima.

Posso contribuir?

Sim. PRs welcome no GitHub. Há um CI automatizado que corre vitest + pytest + Playwright a cada PR — vê o Testing Guide.

Quanto custa?

Grátis e MIT. Self-hosted, sem subscrição. O único custo possível é se escolheres OpenAI API ou ElevenLabs pelos seus preços.
